# Carlow GAA Intensity Analysis

This repository contains a single-page analysis of Carlow's home games against Down, Clare, and Dublin in NHL 1B.

## Files

- `index.html` — interactive scoreboard-style intensity graphs with hover tooltips, dark/light mode, and summary notes.
- `Carlow.csv` — original source file for the Down game intensity data.

## Publish

To publish this page on GitHub Pages:

1. Create a new repository on GitHub and add it as a remote:

```bash
git remote add origin https://github.com/<your-username>/<repo-name>.git
```

2. Push the branch:

```bash
git push -u origin main
```
```

3. Enable GitHub Pages in the repository settings and set the source to the `main` branch.

Once published, the page will be available at `https://<your-username>.github.io/<repo-name>/`.
